ES6 introduced a new method in arrays called map. This is a very useful method, which is used to render lists. So whenever you want to render a list of items, that's when you use the map method of arrays.

Let's say we have to render each and every element in a different colour. So we have array with three colours. So for each color you want to have a list item like this.

const colour = ["Blue","Orange","Green"];
const items = colour.map(colorName => {
    return '<li>'+colorName+'</li> ';
});
We called the map on array colour, color.map. Here we need to pass a callback function. So this function is called by the map method for each item in this array and takes one item at a time transform each element in this array and returns a new item.

So, a callback function takes an item, and returns a new item. Now we take a color which is a string, and simply return a new string like this. Now, this map method returns a new array so it doesn't not modify the original array


This code returns an array with 3 list items which have colours in it => [" <li>Blue</li> ", " <li>Orange</li> ", " <li>Green</li> "]

We can even simplify and make the above code cleaner.

const colour = ["Blue","Orange","Green"];
const items1 = colour.map(colorName => `<li>${colorName}</li>`);
Note we here used something different const items1 = colour.map(colorName => `<li>${colorName}</li>`);. This is called template literals in ES6. So instead of using single or double quotes, we use the back tick character.

Here we can define a template for the string, So we want to have the opening and closing li tags. And in the middle we want to render the color dynamically. So we add a dollar sign and curly braces, this is an argument placeholder, what we put in between the braces will be rendered dynamically at run time. So here we want to render this color a template literal.
