Let us see about objects in this module. We will create an object using const. So as you know objects in JavaScript are collections of key value pairs. So I'm going to add a couple of key value pairs here.
const obj1 = {
    num1: 10, //property
    num2: 8,
    add: function(a,b){ // method
        return a+b;
    },
    subtract(a,b){ //ES6's simpler version of declaring method
        return a-b;
    }
};
There are two ways to access the properties and methods dot notation and bracket notation.

person.add();// dot notation
person['sub'];//bracket notation
