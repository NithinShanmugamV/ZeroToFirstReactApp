We all knew that function is an object made by in-built object called Function(). There is an method for this Function() object called bind which can bind this method to an object.

const opertions = {
    num1: 10,
    num2:8,
    demoPrint() {
        console.log(this) <==
    }
};
const demoPrint= opertions.demoPrint.bind(opertions)
demoPrint();

What did it do? Well bind functions takes a parameter as operations and will return a new instance of this demoPrint function. And set this to point to operations object. So, now when we call this demoPrint function, we will see this person object on the console.
