In modern JavaScript arrow functions are much cleaner and easier way to write a code. Let us take a normal function and look at its arrow function.

const firstName = "Nithin";
const lastName = "Shanmugam";
function greeting1(firstName, lastName) {
    greet = firstName + " " + lastName;
    return greet;
}
let sayHi = greeting1(firstName, lastName);
console.log(sayHi);
const firstName = "Nithin";
const lastName = "Shanmugam";
const greeting2 = (firstName, lastName) => {
    greet = firstName + " " + lastName;
    return greet;
}
let sayHi = greeting2(firstName, lastName);
console.log(sayHi);

If your arrow function takes sigle parameter and has only single code inside the block then it is much easier to write cleaner code. We can get rid of parenthesis.If it doesn't take any parameter add empty braces.

const firstName = "Nithin";
const greeting3 = firstName => "hi " + firstName;
console.log(greeting3(firstName));
const greeting4 = () => "hi ";
console.log(greeting4());
Ok now let us see where this arrow function is helpful. Let us consider an array that has list of students and also their fee paid status. Now write a function that returns list of students who paid the fees.

let studentsPaid = students.filter(function (students){return students.feePaid;});
