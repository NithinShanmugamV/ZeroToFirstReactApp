Spread opertor is also used in cloning an array.

let arr1 = ["Hi","Hello"];
let clone= [...arr1];

Now we can also apply the spread operator on objects to merge two objects.

let obj1 = {
    firstName:"Nithin"
};
let obj2 = {
    secondName:"Shanmugam"
};
let combined = {...obj1,...obj2}
console.log(combined);

Note we used spread operator in {} instead of [] because last time we used array which is represented in square brackets. Now we use objects which is represented in object literal syntax. Also while using spread operator in objects we can also optionally add additional properties to it.
let combined1 = {...obj1,...obj2, age: 20};

similarly we can use spread operator to clone an object
let combined2 = {...combined1};
