Let us now create a class called developer.
class Developer {
    constructor(name,age,role) {
        this.name = name;
        this.age = age;
        this.role = role;
    }
    printRole(){
        console.log(this.role);
    }
}
This has properties name and age which is also a property of class person. In real life applications we may have properties that are also a property of another class.

So, instead of assigning a property again we may inherit property of base class. In this case, we can inherit property name, age of Person to Developer class.


class Developer1 extends person {
    constructor(name,age,role) {
        super(name);
        super(age);
        this.role = role;
    }
    printRole(){
        console.log(this.role);
    }
const dev1 = new Developer1("Nithin",20,"React Developer");
}
We can have this Developer1 class inherit from the person class, and this means it will inherit all the methods defined in this person class.

we add the keyword extends Person. So Developer1 extends person. Now, if we create a Developer1 object dev1, set it to a new Developer1, look how it's constructor. You see the name parameter right? So the Developer1 class is also inheriting the constructor of the person class.

Whenever we add the constructor in a child class we need to call the constructor of it's parent class. So, here in the constructor we have this special keyword, super, that references the parent class.
